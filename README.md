# Numerical Methods – Python Implementations

Collection of numerical algorithms implemented in Python 
as part of university coursework (UMB FPV).

## Implemented Methods

1. Bisection Method
2. Newton Method
3. Jacobi Method
4. Babylonian Method
5. Gauss Elimination
6. Secant Method
7. Lagrange Interpolation
8. Simpson Rule
9. Euler Method
10. Gradient Descent (basic implementation)

## Technologies
- Python 3.x
- Pure mathematical implementation (no heavy libraries)

## Purpose
Educational implementations of core numerical algorithms 
with clear structure and comments.
